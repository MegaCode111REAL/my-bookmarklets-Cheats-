Drag hacks into your bookmarks bar and run them anywhere

📌 Setup (Important)

1. Turn on your browser's Favourites / Bookmarks Bar.

Chrome / Edge: Press Ctrl + Shift + B
Safari: View → Show Favourites Bar
Firefox: View → Toolbars → Bookmarks Toolbar

2. Right click the bar → New Folder

3. Name the folder:

Nothing

4. Drag the cheats below into the Nothing folder.

5. When you're on the site you want, click the bookmark to run the hack.

Tip: You can add as many cheats as you want to the folder.
